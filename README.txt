# QuasiQWERTY #

QuasiQWERTY is a collection of keyboard layouts for macOS.  When Command
or Option are pressed, each layout behaves like the built-in "U.S."
layout, eliminating the need to relearn key combinations for shortcuts
and alternate characters.

## Installation ##

1. Download repository.
2. Copy QuasiQWERTY.bundle to "/Library/Keyboard Layouts" or
   "~/Library/Keyboard Layouts".
3. Log out and back in.
